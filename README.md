# Rust Tutorials

### Installing Rust...

(WSL/*.nix) `curl https://sh.rustup.rs -sSf | sh`

or [just go here](https://www.rust-lang.org/tools/install)

### Manual Installation (with rustup downloaded)

`rustup install <tag>`
`rustup default <tag>`

version tags:
    stable,
    nightly

### Running a project...
`cd directory_name`
`cargo run`

### updating rust
`rustup update`
